📂 **Showcase Repository:** https://github.com/ayrsync/plumberdex-showcase

---

## 📌 Overview

PlumberDex is a modern SaaS platform designed for plumber listings, lead generation, and business management. It enables users to find nearby plumbers while providing plumbers with powerful tools to manage their profiles, services, and customer leads.

---

## ✨ Key Features

- 🔐 Authentication & Role-Based Access (Admin / Plumber / User)
- 📊 Advanced dashboards with analytics
- 📍 Location-based plumber discovery (Google Maps integration)
- 📥 Lead management system
- 🧾 Subscription & plan management
- ⭐ Reviews & ratings system
- 🖼 Gallery & service showcase
- ⚡ SEO-optimized listing pages

---

## 🛠 Tech Stack

### Frontend

- Next.js 16
- TypeScript
- Tailwind CSS
- ShadCN UI

### Backend

- NestJS
- Prisma ORM
- PostgreSQL (with PostGIS for location)

### Integrations

- Google Maps API (location & autocomplete)
- Stripe / Razorpay (payments)
- JWT Authentication
